Longest Palindrome Implement a function that finds the longest palindrome in a given stringing. For example, in the stringing “My dad is a racecar athlete”, the longest palindrome is “a racecar a”. Count whitespaces as valid characters. Other palindromes in the above string include “dad”, “ete”, “ dad “ (including whitespace on each side of dad).
